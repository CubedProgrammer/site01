Fibonacci

Fibonacci has one male rabbit and one female rabbit. Rabbits produce one pair of offspring each month, one male and female, and takes one month to grow up.

Rabbits will inbreed, Fibonacci starts off with one pair of rabbits, one male and one female.

The Fibonacci sequence is the sequence of number of pairs of adult rabbits in each month.

Month Adults Babies
1 1 0
2 1 1
3 2 1
4 3 2
5 5 3
6 8 3

The Fibonacci sequence can be written as a function over the integers, defined recursively.

f(1)=1, f(2)=1, f(n)=f(n-1)+f(n-2)

This formula can sequentially generate the next number in the sequence. However, if intermediate values are not desired, there is a much faster way to compute the value at a specific index.

f(a)f(b-1)+f(a+1)f(b)=f(a+b)

This allows us to double the index of the sequence every iteration of the formula, by setting a equal to b.

f(a)f(a-1)+f(a+1)f(a)=f(2a)

As we can see, the formula requires three smaller values to generate one value, to continuously generate values. Here are formulas to generate values efficiently.

f(a-1)2+f(a)2=f(2a-1)

2f(a-1)f(a)+f(a)2=f(2a)

We skipped the third value because it is not actually necessary. We only need two values.

For when a is not a power of two, resort back to the original formula.
