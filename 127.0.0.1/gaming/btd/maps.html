BTD6 Custom Maps

BTD6 allows the creation of maps using the custom map editor, the maps can then be published for other players to play. You must beat your own map before publishing it, here is a list of my custom maps.

All of the following maps have been tested to be possible on CHIMPS mode. That was the game mode played in order for the maps to be published in the first place.
