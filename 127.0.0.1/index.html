Meaning of the Name

A polyhedron is a shape in three dimensions with flat polygonal faces, straight edges, and sharp corners. A hexahedron specifically has six faces, such as a cube. A hexahedron always has four more edges than vertices.

A regular polyhedron have all faces congruent, all edges equal, and all vertices equal.

Useful Programs

These programs may be useful to you, especially those of you who are programmers. They are all on GitHub, and bugs should be reported to the GitHub issue page.

linux_utility

A collection of utilities for linux, all of whom can be executed just on the terminal. They should also work on other POSIX-compliant systems. Some of them will even compile and run on windows.

csloc

A program to count how many lines of code is in all files inside a certain directory, can filter by file extension. It can also list total count by extension, or sort by lines of code.

CProgramBuilder

A build system for C and C++17 projects on POSIX systems. Smaller and easier to set up than make, and automatically detects header dependencies. Supports GCC and Clang.

C++ProgramBuilder

A build system for C++20 projects on POSIX systems. Compiles external modules automatically, and automatically adds relevant compiler flags for modules. Supports GCC and Clang.
